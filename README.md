# Personal Portfolio Website

## Objective
This project is a responsive personal portfolio website developed using HTML, CSS, Bootstrap, and JavaScript.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features
- Responsive Design
- Smooth Scrolling
- Bootstrap Navigation Bar
- Home, About, Skills, Projects & Contact Sections
- Scroll-to-Top Button
